## Что умеет

Открываешь любой проект Behance — BeRaw сканирует страницу, обводит большие картинки синим и кладёт в боковую панель, чтобы ты выбрал. По одной или «Выбрать все», отдельными файлами или одним ZIP.

- **Сырые значит сырые** — BeRaw тащит самый большой файл с CDN Behance, а не уменьшенное превью из ленты.
- **Кнопка на каждом изображении** — В оверлее картинки есть кнопка, чтобы скачать её одну в выбранном формате.
- **Пакетный ZIP** — Панель упаковывает всю выборку в один ZIP. На Windows не надо нажимать «Сохранить как» для каждого файла.
- **Контроль формата** — Оригинальные байты, WebP→JPG автоматом, всё в JPG, или всё в PNG.
- **12 языков** — упрощённый/традиционный китайский, English, 日本語, 한국어, Français, Deutsch, Español, Português, Русский, Italiano, Tiếng Việt.

## Установка

### Из GitHub Releases (рекомендуется)

1. Скачай последний `.zip` со [страницы Releases](https://github.com/hooosberg/BeRaw/releases/latest).
2. Распакуй в стабильное место (не удаляй папку — Chrome загружает расширение оттуда).
3. Открой `chrome://extensions/` и включи **Режим разработчика** справа вверху.
4. Нажми **Загрузить распакованное расширение** и выбери распакованную папку.
5. Открой проект Behance — появятся синие рамки.

Встроенная проверка обновлений (Настройки → О программе → **Проверить обновления**) использует тот же API, ты увидишь новые версии сразу.

### Из исходников (для контрибьюторов)

```bash
git clone https://github.com/hooosberg/BeRaw.git
```

В `chrome://extensions/` включи режим разработчика, нажми **Загрузить распакованное расширение** и выбери `local/`.

### Из Chrome Web Store

[Установить из Chrome Web Store](https://chromewebstore.google.com/detail/beraw-%E2%80%94-behance-raw-image/ghhedamclfpgcnpfclmefepnacnkngio?authuser=0&hl=en).

## Использование

| Шаг | Действие |
| --- | -------- |
| 1 | Открой страницу проекта Behance |
| 2 | Дождись синих рамок (большие картинки, найденные BeRaw) |
| 3 | Нажми **Выбрать** на картинке, или **Выбрать все** в панели |
| 4 | Выбери формат: **Оригинал · Авто · JPG · PNG** |
| 5 | **Скачать выбранное (ZIP)** для всего, или **Скачать изображение** в оверлее одной |

## Форматы вывода

| Формат | Поведение |
| ------ | --------- |
| **Оригинал** | Без перекодирования. Самый большой файл с CDN Behance. WebP перекодируется в JPG/PNG с максимальным качеством. |
| **Авто** | Только WebP становится JPG, остальное как есть. |
| **JPG** | Всё в JPG, прозрачные области белым. Удобно для сдачи. |
| **PNG** | Всё в PNG, прозрачность сохраняется. |
